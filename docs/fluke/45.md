# Fluke 45 Dual Display Multimeter Programming Context

## 1. Communication Interface Setup

The Fluke 45 can be operated remotely via **RS-232** (standard) or **IEEE-488/GPIB** (optional). Only one interface can be enabled at a time.

### RS-232 Interface

*
**Connector:** DB-9 (male pins) on the rear panel.


* **Parameters (Factory Defaults):**
* Baud Rate: 9600


* Parity: None (8 data bits)


* Stop Bits: 1


* Echo: On




* **Prompts:**
* `=>`: Command successfully executed.
* `?>`: Command Error (syntax error).
*
`!>`: Execution Error (device-dependent error) .




*
**Device Clear:** Send `^C` (Control-C) to clear the device buffer (equivalent to IEEE-488 DCL).



### IEEE-488 (GPIB) Interface

* **Enabling:** Must be selected from the front panel. Press `2ND` + `RATE` (BAUD) and scroll to select "IEEE".


* **Address:** Set from the front panel. Press `2ND` + `MN MX` (ADDR) and scroll to select an address (0-30) .


*
**Limitations:** Max 15 devices, max cable length 20 meters .



---

## 2. Command Syntax & Processing

* **Input Terminators:**
* RS-232: `<CR>`, `<LF>`, or `<CR><LF>` .


* IEEE-488: EOI on any character, or `<LF>`.




*
**Execution Rule:** The meter executes commands sequentially (first-in, first-out).


* **Output Buffer:** Must be read once per query. Reading it twice without an intervening query causes the meter to hang or not respond.


*
**Input Buffer:** 350 bytes.

## 3. IEEE 488.2 Common Commands

| Command | Description | Return Format |
| --- | --- | --- |
| `*IDN?` | Identification Query. Returns manufacturer, model, serial, and firmware. | <br>`FLUKE, 45, <Serial>, <MainVer> <DispVer>`.

 |
| `*RST` | Reset. Restores power-up configuration. | None.

 |
| `*CLS` | Clear Status. Clears event registers. | None.

 |
| `*TRG` | Trigger. Triggers a measurement. | None.

 |
| `*TST?` | Self-Test. Runs internal diagnostics (approx 15s). | <br>`0` (Pass) or error code (e.g., `1` for ROM fail).

 |
| `*STB?` | Read Status Byte. | Integer representing the status byte.

 |
| `*ESR?` | Event Status Register Query. Returns and clears ESR. | Integer.

 |

## 4. Measurement Function Commands

These commands select the measurement function for the **Primary** (1) or **Secondary** (2) display.

| Primary Cmd | Secondary Cmd | Description |
| --- | --- | --- |
| `VAC` | `VAC2` | AC Volts.

 |
| `VDC` | `VDC2` | DC Volts.

 |
| `VACDC` | N/A | AC + DC RMS Volts (Primary only).

 |
| `AAC` | `AAC2` | AC Current.

 |
| `ADC` | `ADC2` | DC Current.

 |
| `OHMS` | `OHMS2` | Resistance.

 |
| `FREQ` | `FREQ2` | Frequency.

 |
| `DIODE` | `DIODE2` | Diode Test.

 |
| N/A | `CLR2` | Clears the secondary display.

 |

---

## 5. Range and Rate Commands

* **Auto Range:** `AUTO` enables autoranging. `AUTO?` returns `1` (on) or `0` (off) .


* **Manual Range:** `RANGE <range_code>` sets a fixed range (1-7).
* *Example:* `RANGE 1` selects 300mV/300Ω/30mA. `RANGE 4` selects 300V/300kΩ.




* **Measurement Rate:** `RATE <speed>`
* `S` = Slow (2.5 rdg/s, 5 digits).
* `M` = Medium (5 rdg/s, 4.5 digits).
*
`F` = Fast (20 rdg/s, 3.5 digits).





---

## 6. Measurement Queries

To retrieve data from the meter.

| Query | Description |
| --- | --- |
| `val1?` | Returns value from Primary Display.

 |
| `val2?` | Returns value from Secondary Display.

 |
| `meas1?` | Triggers and returns Primary Display value.

 |
| `meas2?` | Triggers and returns Secondary Display value.

 |
| `meas?` | Triggers and returns values from both displays (comma-separated).

 |

## 7. Function Modifiers

Modifiers alter the reading or comparison logic.

*
**Relative Mode:** `REL` (enters relative mode), `RELSET <value>` (sets offset), `RELCLR` (exits).


* **Decibels:** `DB` (enters dB mode), `DBCLR` (exits). `DBREF <value>` sets reference impedance (e.g., 50, 75, 600).


*
**Hold:** `HOLD` (enters Touch Hold), `HOLDCLR` (exits).


*
**Min/Max:** `MIN`, `MAX` (displays stored min/max), `MMCLR` (clears/exits).


*
**Compare:** `COMP` (enters compare mode), `COMPHI <value>`, `COMPLO <value>` (set limits), `COMP?` (returns HI, LO, PASS).



---

## 8. Trigger Configuration

Control when measurements are taken.

* **Command:** `TRIGGER <type>`
* `1`: Internal (Continuous).
* `2`: External (Rear disabled, no delay).
* `3`: External (Rear disabled, delay enabled).
* `4`: External (Rear enabled, no delay).
* `5`: External (Rear enabled, delay enabled).


*
**Triggering:** Send `*TRG` over the bus to trigger a measurement when in external trigger mode.



---

## 9. Example SCPI Sequences

* **Set DC Volts, Auto Range, Medium Rate:**
`VDC; AUTO; [cite_start]RATE M`.


* **Measure Frequency (Primary) and AC Volts (Secondary):**
`FREQ; VAC2; MEAS?`
* **Set Manual Range 300V:**


`RANGE 4`.


* **Read Primary Display Value:**


`VAL1?`.
