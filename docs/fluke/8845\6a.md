# Fluke 8845A/8846A Programmer's Context

## 1. Remote Interface Configuration

The meter can be controlled via RS-232, IEEE 488 (GPIB), or Ethernet (LAN). Only one interface is active at a time, selected via the front panel .

### RS-232 Configuration

* **Parameters:** Baud rate (1200-230400), Parity (None/Odd/Even), Data bits (7/8), Stop bits (1/2), Flow control (None/XON/RTS).
*
**Default Settings:** 9600 baud, No parity, 8 data bits, 1 stop bit, No flow control.


* **Modes:**
*
**884X Mode:** Has "Terminal" (interactive, echoes characters) and "Computer" (no echo, prompts suppressed) modes .


*
**Fluke 45 Mode:** Has "Echo" and "No Echo" modes .




*
**Cable:** Null modem cable (e.g., Fluke RS43) required for PC connection.



### IEEE 488 (GPIB) Configuration

*
**Address:** 1-32 (Default: 1), set via front panel .


*
**Capabilities:** SH1, AH1, T5, L4, SR1, RL1, PP0, DC1, DT1, C0, E2.



### Ethernet (LAN) Configuration

*
**Connection:** Telnet to port 3490 (default).


*
**IP Address:** Supports DHCP (default) or Static IP .


*
**Default IP:** `129.196.136.131`.

## 2. Command Set Selection

The meter supports two command sets, selectable via `SYST:LANG` or front panel:

*
**8845 (SCPI):** Native mode, compatible with Agilent 34401A.


*
**Fluke 45:** Emulation mode for legacy compatibility.

## 3. SCPI Command Summary (Native Mode)

### Measurement Commands (`MEASure?`)

Simplest method to configure and read a measurement in one step.

* **Syntax:** `MEASure[:SCALar]:<function>? [<range>, <resolution>]`
*
**Functions:** `VOLT:DC`, `VOLT:AC`, `CURR:DC`, `CURR:AC`, `RES`, `FRES` (4-wire), `FREQ`, `PER`, `CAP` (8846A), `TEMP` (8846A), `DIOD`, `CONT` .


* **Example:** `MEAS:VOLT:DC? 10, 0.0001` (Measure DC Volts, 10V range, 100uV resolution).

### Configuration Commands (`CONFigure`)

Presets parameters without initiating measurement. Used with `READ?` or `INIT` + `FETCH?`.

* **Syntax:** `CONFigure[:SCALar]:<function> [<range>, <resolution>]`
* **Parameters:** Same functions as `MEASure?`.
* **Example:** `CONF:RES 1000` (Configure 2-wire resistance, 1k range).

### Triggering & Reading Data

*
**`INITiate`:** Puts meter in "wait-for-trigger" state.


*
**`READ?`:** Triggers measurement and returns value (equivalent to `INIT` + `FETCH?`).


*
**`FETCh?`:** Retrieves measurements from internal memory to output buffer.


*
**`TRIGger:SOURce`:** `IMM` (Immediate/Internal), `EXT` (External hardware), `BUS` (*TRG command).


*
**`TRIGger:DELay`:** 0 to 3600 seconds.


*
**`SAMPle:COUNt`:** 1 to 50,000 readings per trigger.



### Sense Subsystem (Detailed Configuration)

* **Voltage/Current:**
*
`[SENSe:]VOLT:DC:NPLC <value>`: Set integration time (0.02, 0.2, 1, 10, 100 PLC).


*
`[SENSe:]VOLT:AC:BAND <value>`: Set AC filter (3, 20, 200 Hz).


*
`[SENSe:]DET:BAND <value>`: Set global AC filter.


*
`[SENSe:]ZERO:AUTO <ON|OFF|ONCE>`: Control autozero.


*
`[SENSe:]INPut:IMPedance:AUTO <ON|OFF>`: Enable high input Z (>10GÎ©) for low DC ranges.




* **Resistance:**
*
`[SENSe:]RES:NPLC <value>`: Set NPLC for resistance.




* **Temperature (8846A):**
*
`[SENSe:]TEMP:RTD:TYPE <type>`: `PT100_385`, `PT100_392`, `CUST1`.


*
`[SENSe:]TEMP:RTD:R0 <value>`: Set R0 resistance.





### Math Functions (`CALCulate`)

*
**Function Selection:** `CALC:FUNC <NULL|DB|DBM|AVER|LIM>`.


*
**Enable:** `CALC:STAT ON`.


* **Operations:**
*
**Null:** `CALC:NULL:OFFS <value>`.


*
**dB/dBm:** `CALC:DB:REF <value>`, `CALC:DBM:REF <value>`.


*
**Statistics:** `CALC:AVER:MIN?`, `CALC:AVER:MAX?`, `CALC:AVER:AVER?` .


*
**Limits:** `CALC:LIM:LOW <value>`, `CALC:LIM:UPP <value>`.


*
**mX+b:** `CALC:KMAT:MMF <value>` (m), `CALC:KMAT:MBF <value>` (b).





### System & Status

*
**`SYSTem:ERRor?`:** Read and clear error queue.


*
**`*IDN?`:** Identification string.


*
**`*RST`:** Reset instrument.


*
**`DATA:POINts?`:** Count stored readings.

## 4. Fluke 45 Emulation Commands

Compatible commands for legacy Fluke 45 programs.

### Function Commands

* `VAC`, `VDC`: AC/DC Volts.
* `AAC`, `ADC`: AC/DC Current.
* `OHMS`: Resistance.
* `FREQ`: Frequency.
* `DIODE`: Diode Test.
*
`VAC2`, `VDC2`, etc.: Secondary display functions.



### Modifiers & Queries

* `DB`, `DBCLR`, `DBPOWER`.
* `VAL1?`, `VAL2?`: Primary/Secondary values.
* `MEAS1?`, `MEAS2?`: Trigger and read values.
*
`MEAS?`: Read both displays.



### Incompatibilities

*
**Missing Features:** `AACDC`, `VACDC`, `HOLD` (Touch Hold), `MIN/MAX` (use Math instead).


* **Range Diff:** Fluke 45 ranges differ slightly; 884XA maps them to nearest equivalent.


*
**Triggering:** Rear trigger is enabled for types 1, 2, 3 in emulation mode (unlike original Fluke 45).



---

## 5. Status Reporting

Standard status register structure (IEEE 488.2).

* **Status Byte (STB):** Summary of other registers. Read with `*STB?`.
* **Standard Event (ESR):** Specific events (Command Error, Execution Error, etc.). Read with `*ESR?`.
* **Questionable Data:** Overload conditions, limit failures. Read with `STAT:QUES:EVEN?`.


*
**Output Buffer:** Stores query responses.



---

## 6. Error Codes

* **-1xx:** Command/Syntax errors (e.g., `-115` Missing parameter).
* **-2xx:** Execution errors (e.g., `-222` Data out of range, `-213` Init ignored).
* **-3xx:** System errors (e.g., `-350` Queue overflow).
*
**5xx:** Interface errors (RS-232/GPIB/LAN specific) .
