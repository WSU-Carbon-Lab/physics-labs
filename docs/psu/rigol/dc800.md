# Rigol DP800 Series Programming Command Context

## 1. Device: Rigol DP800 Series Power Supply

### Communication & Syntax

*
**Interfaces:** USB, LAN, RS232, GPIB (optional) .


* **Terminators:**
*
**Command:** New line character `<NL>` (0x0A) .




* **Execution:** Sequential execution of commands.
*
**Command Format:** Colon-separated SCPI commands (e.g., `:APPLy CH1,5,1`) .


* **Parameter Types:**
*
`<value>`: Numeric value (e.g., `1.5`, `MAX`, `MIN`) .


*
`<state>`: Boolean (e.g., `ON`, `OFF`, `1`, `0`) .




*
**Queries:** Append a `?` to the command (e.g., `:APPLy?`).



### IEEE 488.2 Common Commands

| Command | Description | Return Format |
| --- | --- | --- |
| `*IDN?` | Identification Query | <br>`Rigol Technologies,<Model>,<Serial>,<Firmware>`

 |
| `*RST` | Reset to factory default | None

 |
| `*CLS` | Clear Status | None

 |
| `*TRG` | Trigger command | None

 |
| `*TST?` | Self-Test Query | <br>`0` (Pass) or error code

 |
| `*OPC?` | Operation Complete Query | <br>`1`

 |
| `*WAI` | Wait for Operation Complete | None

 |
| `*SAV <n>` | Save current state | None

 |
| `*RCL <n>` | Recall saved state | None

 |

### Output Configuration Commands

These commands configure the output parameters (voltage and current) for specific channels.

| Command | Description | Parameters |
| --- | --- | --- |
| `:APPLy <ch>,<volt>,<curr>` | Select channel and set volt/curr | `CH1

 |
| `:APPLy? <ch>` | Query voltage/current setting | Returns `<volt>,<curr>`

 |
| `:OUTPut <ch>,<state>` | Enable/Disable output | `CH1

 |
| `:OUTPut:TRACk <ch>,<state>` | Enable/Disable track mode | <br>`CH1`, `ON |
| `:OUTPut:OVP:VALue <ch>,<val>` | Set OVP level | `CH1

 |
| `:OUTPut:OCP:VALue <ch>,<val>` | Set OCP level | `CH1

 |
| `:OUTPut:OVP <ch>,<state>` | Enable/Disable OVP | `CH1

 |
| `:OUTPut:OCP <ch>,<state>` | Enable/Disable OCP | `CH1

 |

### Measurement Commands

Use these commands to read the actual output values.

| Command | Description | Return Format |
| --- | --- | --- |
| `:MEASure:VOLTage? <ch>` | Measure voltage | Voltage value (e.g., `5.000`)

 |
| `:MEASure:CURRent? <ch>` | Measure current | Current value (e.g., `1.000`)

 |
| `:MEASure:POWer? <ch>` | Measure power | Power value (e.g., `5.000`)

 |
| `:MEASure:ALL? <ch>` | Measure volt, curr, power | <br>`Voltage,Current,Power`

 |

### Source Configuration Commands

Fine-tune voltage and current settings.

| Command | Description | Parameters |
| --- | --- | --- |
| `:SOURce:VOLTage <val>` | Set voltage level | Value

 |
| `:SOURce:CURRent <val>` | Set current limit | Value

 |
| `:SOURce:VOLTage:PROTection <val>` | Set OVP level (Source subsystem) | Value

 |
| `:SOURce:CURRent:PROTection <val>` | Set OCP level (Source subsystem) | Value

 |

### System & Utility Commands

*
**Beeper:** `:SYSTem:BEEPer <state>`.


*
**Language:** `:SYSTem:LANGuage <lang>` (e.g., `SCPI`) .


*
**Lock Front Panel:** `:SYSTem:LOCK <state>`.


*
**Save/Recall:** `:MEMory:STATe:SAVE <n>`, `:MEMory:STATe:RECall <n>` .



### Triggering

*
**Trigger Source:** `:TRIGger:SOURce <source>` (`IMM`|`BUS`|`EXT`) .


*
**Trigger Delay:** `:TRIGger:DELay <val>` .


*
**Initiate Trigger:** `:INITiate`.



### Timer & Delay Configuration (Programmable Output)

*
**Timer Enable:** `:TIMer:STATe <state>` .


*
**Timer Parameters:** `:TIMer:PARAmeter <n>,<volt>,<curr>,<time>` .


*
**Delay Enable:** `:DELayer:STATe <state>` .


*
**Delay Parameters:** `:DELayer:PARAmeter <n>,<state>,<time>` .



### Monitor & Analyzer

*
**Monitor Enable:** `:MONItor:STATe <state>` .


*
**Analyzer Enable:** `:ANALyzer:STATe <state>`.

## 3. Programming Examples

### Python (PyVISA) Setup

```python
import pyvisa as visa

rm = visa.ResourceManager()

# Open connection via USB
# Replace <serial_number> with actual device serial
device = rm.open_resource('USB0::0x1AB1::0x0E11::<serial_number>::INSTR')

# Configure Output 1 to 5V, 1A
device.write(':APPLy CH1,5,1')

# Enable Output 1
device.write(':OUTPut CH1,ON')

# Measure Output
voltage = device.query(':MEASure:VOLTage? CH1')
current = device.query(':MEASure:CURRent? CH1')

print(f"Voltage: {voltage} V, Current: {current} A")

# Turn Off Output
device.write(':OUTPut CH1,OFF')

device.close()

```
